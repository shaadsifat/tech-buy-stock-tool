Other Link Upload

Products themselves only come from the Shopify catalog sync now — there's no manual one-by-one add anymore. This page only attaches/updates the Other-site link on a product that already exists, matched by its Shopify Handle (pulled straight out of the Tech Buy Link column — never by name, and not case-sensitive).

Bulk Upload

Upload an Excel (.xlsx) file to set the Other-site link on many existing products at once. The first row must be headers, in this exact order:

Category Product Name Tech Buy Link Other website link

The product is matched by the handle in the Tech Buy Link column — Category and Product Name are only used for your own reference and in the failure report below, never for matching. A Tech Buy Link whose handle doesn't already exist in the database can't be added this way.
